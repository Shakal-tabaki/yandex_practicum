Название: Оптимизация производственных расходов. 
Определение темперературы розлива ковшей со сплавом.

Описание: Металлургический комбинат ООО «Так закаляем сталь» решил уменьшить потребление электроэнергии на этапе обработки стали. 
Данный проект направлен на реализацию поставленной задачи, 
чтоб свою очередь подразумевает постройку модели способной предсказать температуру стали, на предоставленных данных.

ЯП: Python 3.10.5

Дополнительно: Будут испытаны другие интрументы ML на предоставленных данных. 
А также выбрать другую метрику при оценке качетсва.
